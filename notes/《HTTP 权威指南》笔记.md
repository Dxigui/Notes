## 《HTTP 权威指南》



HTTP 方法

| HTTP 方法 | 描述                                           |
| --------- | ---------------------------------------------- |
| GET       | 从服务器向客户端发送命名资源                   |
| PUT       | 将来自客服端的数据存储到一个命名服务器资源中去 |
| DELETE    | 从服务器中删除命名资源                         |
| POST      | 将客户端数据发送到一个服务器网关应用程序       |
| HEAD      | 仅发送命名资源响应中的 HTTP 首部               |



HTTP 报文: 请求报文/响应报文

包含三部分:

* 起始行
* 首部字段
* 主体
